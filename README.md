# Blend × Whish — Developer Docs

Official Blend developer documentation for selling Blend event tickets natively
inside the Whish app. Single-page static site (no build step).

- **Live:** deployed on Vercel → intended custom domain `docs.blendapp.ai`
- **Source of the full reference:** `docs/partners/whish/` in the Blend monorepo
- **Maintained by:** Blend Engineering · developers@blendapp.ai

## Deploy

Static — Vercel serves `index.html` directly. Push to the default branch to
deploy. To attach the custom domain, add `docs.blendapp.ai` in the Vercel project
and point a CNAME at Vercel in Cloudflare DNS.
